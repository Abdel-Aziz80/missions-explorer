# Missions Explorer (démo)

Dashboard React consommant une API REST.  
Projet de démonstration orienté **mission B2B / prestation de développement web**.

## Fonctionnalités
- Récupération de données via API REST
- Recherche dynamique
- Filtres par tags
- Tri des résultats
- États loading / erreur
- Modal de détail
- Interface responsive

## Stack
- React
- JavaScript (ES6+)
- Vite
- API REST (DummyJSON)

## Objectif
Démontrer la capacité à développer une application front-end
structurée, orientée données et **besoins métier**,
dans un contexte de **prestation ou mission de renfort**.

Ce projet illustre une capacité à intervenir sur des applications existantes
dans un contexte professionnel structuré.

## Informations
Données fictives — projet de démonstration publique.
